This repository contains implementations of various sorting algorithms in C++.
These algorithms are fundamental for understanding data structures and algorithms, and are essential for efficient data manipulation.

Included Algorithms
•	Selection Sort
•	Bubble Sort
•	Insertion Sort
•	Merge Sort
•	Quick Sort
•	Counting Sort
•	Heap Sort

Time and Space Complexities
1. Selection Sort
•	Time Complexity:
o	Best: O(n²)
o	Average: O(n²)
o	Worst: O(n²)
•	Space Complexity: O(1)
2. Bubble Sort
•	Time Complexity:
o	Best: O(n)
o	Average: O(n²)
o	Worst: O(n²)
•	Space Complexity: O(1)
3. Insertion Sort
•	Time Complexity:
o	Best: O(n)
o	Average: O(n²)
o	Worst: O(n²)
•	Space Complexity: O(1)
4. Merge Sort
•	Time Complexity:
o	Best: O(n log n)
o	Average: O(n log n)
o	Worst: O(n log n)
•	Space Complexity: O(n)
5. Quick Sort
•	Time Complexity:
o	Best: O(n log n)
o	Average: O(n log n)
o	Worst: O(n²)
•	Space Complexity: O(log n)
6. Counting Sort
•	Time Complexity:
o	Best: O(n + k)
o	Average: O(n + k)
o	Worst: O(n + k)
•	Space Complexity: O(k)
7. Heap Sort
•	Time Complexity:
o	Best: O(n log n)
o	Average: O(n log n)
o	Worst: O(n log n)
•	Space Complexity: O(1)

Usage
1.	Choose the sorting algorithm you want to test.
2.	Follow the instructions in the code to input your data.
3.	View the sorted output.
